## SM-2 算法核心

### 参数定义
- **n**：复习次数（0=新卡片，1=第一次复习...）
- **EF**：简易度因子（Easiness Factor，初始 2.5）
- **I**：间隔天数（Interval）

### 评分标准
| 评分 | 含义 | 下次间隔 |
|------|------|----------|
| 0 | 完全忘记 | 1天 |
| 1 | 困难回忆 | 1天 |
| 2 | 模糊记得 | 2天 |
| 3 | 记得 | 间隔 × EF |
| 4 | 秒答 | 间隔 × EF × 1.3 |

### 算法公式
```
if 评分 < 3:
    间隔 = 1
    n = 0
else:
    if n == 0: 间隔 = 1
    if n == 1: 间隔 = 6
    if n > 1: 间隔 = 上次间隔 × EF
    n = n + 1

EF = EF + (0.1 - (5-评分) × (0.08 + (5-评分) × 0.02))
if EF < 1.3: EF = 1.3
```

## ADHD 友好设计

### 防过载机制
- **每日上限**：最多 10 张卡片
- **时间盒**：单次复习不超过 10 分钟
- **中断友好**：可随时暂停，下次继续

### 即时反馈
- 每张卡片完成后立即显示结果
- 连续正确有视觉奖励
- 困难卡片标记，降低优先级

## 卡片来源

- 知识管理引擎生成的永久笔记
- 用户手动添加的问答对
- 重要概念、定义、方法论

## 与知识管理的闭环

1. 知识管理生成卡片 → 进入复习队列
2. 复习中表现好的卡片 → 归档
3. 复习中表现差的卡片 → 标记需重新学习
4. 用户反馈 → 优化卡片质量
