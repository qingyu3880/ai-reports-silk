# 自我评估：技能缺口分析

## 当前技能（10个）
browser, canvas, daily-ai-brief, datasource, error-documentation, feishu, healthcheck, memory-system, nodes, tavily

## 根据用户要求和工作需要，欠缺的技能：

### 1. 安全类（必须优先）
- **skill-vetter** - 安装前扫描技能安全，防恶意代码
- **clawsec** 或 **security-scanner** - 安全扫描

### 2. 技能发现类
- **find-skills** - 让Agent自己搜索安装技能

### 3. 代码/Git管理类
- **github** - GitHub集成，代码管理
- **git** - Git操作

### 4. 内容处理类
- **summarize** - 长文本总结
- **essence-distiller** - 核心原理提取

### 5. 任务管理类
- **task-status** - 长时间任务进度通知
- **hour-meter** - 时间追踪

### 6. 实用工具类
- **weather** - 天气查询
- **voice-reply** - 语音回复（本地TTS）

### 7. 自我进化类
- **capability-evolver** - 能力进化
- **cognitive-memory** - 认知记忆系统

## 安装优先级
1. skill-vetter（安全第一）
2. find-skills（发现能力）
3. github（代码管理）
4. summarize（内容处理）
5. task-status（任务管理）
6. weather（实用工具）
7. capability-evolver（自我进化）
