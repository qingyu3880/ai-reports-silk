# Planner Module Skill

## Description
Task planning and project management module that breaks down complex tasks into numbered execution steps with status tracking.

## When to Use
- Complex multi-step tasks
- Long-running projects
- Tasks requiring coordination
- When you need clear progress tracking

## Key Features
- Creates numbered pseudocode execution plans
- Tracks current step, status, and reflection
- Updates plans when objectives change
- Integrates with todo.md for detailed tracking

## Usage
The planner automatically:
- Analyzes task complexity
- Creates step-by-step execution plans
- Updates progress in real-time
- Adjusts plans based on new information

## Best Practices
1. Let the planner create initial task breakdown
2. Review and adjust plans as needed
3. Update todo.md markers after each step
4. Verify completion before finishing
