# Skill: Summarize

## Description
长文本快速总结工具。把长文章、报告、会议纪要压缩成精华。

## When to Use
- 阅读长篇文章时提取重点
- 处理堆积的邮件和文档
- 快速了解行业报告核心内容
- 生成会议纪要摘要

## Key Features
- 多格式支持：文本、PDF、网页
- 可调节摘要长度
- 保留关键数据和结论
- 支持批量处理

## Usage Examples
- "总结这篇长文章"
- "把这份PDF报告压缩成3段话"
- "提取这封邮件的核心要点"
- "生成今天的会议纪要摘要"

## Installation
```bash
npx clawhub@latest install summarize
```

## Note
这是一个基于提示词的技能，核心能力是文本分析和提炼。
