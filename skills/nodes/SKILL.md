# Nodes Skill

## Description
Manage and control paired nodes/devices for distributed operations. Remote device management and monitoring.

## When to Use
- Managing multiple devices
- Remote operations
- Distributed task execution
- IoT device control

## Key Features
- Node discovery and pairing
- Remote command execution
- Device status monitoring
- Cross-device coordination

## Usage
- List connected nodes
- Execute commands on remote devices
- Monitor device status
- Coordinate multi-device workflows

## Best Practices
1. Verify node connectivity first
2. Use secure communication channels
3. Monitor device resources
4. Handle node failures gracefully
