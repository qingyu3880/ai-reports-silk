# Knowledge Module Skill

## Description
Best practices and domain knowledge reference system. Provides task-relevant knowledge and expertise through event stream.

## When to Use
- Need domain expertise on unfamiliar topics
- Looking for best practices
- Want to avoid common pitfalls
- Need contextual guidance

## Key Features
- Task-relevant knowledge via event stream
- Scoped knowledge items (only use when conditions met)
- Best practice references
- Domain expertise on demand

## Usage
The knowledge module automatically:
- Identifies relevant knowledge for current task
- Provides contextual best practices
- Warns about common mistakes
- Suggests optimal approaches

## Best Practices
1. Review knowledge items when provided
2. Apply knowledge only when conditions match
3. Use knowledge to supplement, not replace, thinking
4. Cross-reference multiple knowledge items
