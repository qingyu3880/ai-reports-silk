# Ontology Skill

## Description
Knowledge graph system that stores information as typed entities and relationships, enabling structured memory and relationship reasoning.

## When to Use
- Project management with complex relationships
- Knowledge management requiring structured data
- Long-term memory with type validation
- Relationship inference and querying

## Key Features
- Typed entities (Person, Company, Project, Task)
- Relationship mapping (works_for, has_task, depends_on)
- Natural language triggers
- Constraint validation

## Usage Examples
- "Remember Alice works on Project X"
- "What depends on this task?"
- "Show all team members for Project Y"
