# Skill: Task Status

## Description
长时间任务进度通知工具。后台任务运行时定期推送进度摘要。

## When to Use
- 执行长时间运行的任务
- 需要持续监控进度
- 团队协作时同步状态
- 防止任务"黑盒"运行

## Key Features
- 定期进度报告（每5/10/30分钟）
- 简短清晰的状态描述
- 支持团队同步
- 任务完成通知

## Usage Examples
- "执行这个长时间任务，每10分钟汇报一次进度"
- "监控这个数据处理任务的状态"
- "团队同步：报告当前任务进展"

## Installation
```bash
npx clawhub@latest install task-status
```

## Best Practices
1. 设置合理的汇报间隔
2. 进度描述要简洁明了
3. 异常情况立即报告
4. 任务完成时总结成果
