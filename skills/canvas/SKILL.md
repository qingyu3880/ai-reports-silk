# Canvas Skill

## Description
Visual canvas for presenting information, creating diagrams, and interactive displays. Alternative to text-only responses.

## When to Use
- Need visual presentation
- Creating diagrams or charts
- Building interactive displays
- Visualizing complex data

## Key Features
- HTML/CSS/JS canvas rendering
- Image generation and display
- Interactive elements
- Screenshot capabilities

## Usage
- Present data visually
- Create diagrams
- Build simple interactive tools
- Render charts and graphs

## Best Practices
1. Use for complex visualizations
2. Keep designs clean and readable
3. Test interactivity before sharing
4. Provide text alternatives when needed
