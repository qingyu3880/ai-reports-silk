# Feishu (Lark) Suite Skill

## Description
Complete Feishu integration for document collaboration, file management, and knowledge base operations.

## When to Use
- Writing to Feishu documents
- Managing cloud files
- Accessing Feishu Wiki knowledge base
- Team collaboration workflows

## Components
- Feishu Doc: Read/write documents
- Feishu Drive: File management
- Feishu Wiki: Knowledge base access

## Usage Examples
- "Create a Feishu document with this analysis"
- "Upload this file to Feishu Drive"
- "Search the company wiki for onboarding info"
