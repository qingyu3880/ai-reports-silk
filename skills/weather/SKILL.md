# Skill: Weather

## Description
天气查询技能，支持全球城市实时天气和3天预报。

## When to Use
- 查询当前天气
- 获取未来几天预报
- 出行前查看天气
- 安排户外活动

## Key Features
- 全球城市支持
- 实时天气数据
- 3天预报
- 中文查询友好

## Usage Examples
- "上海明天天气怎么样"
- "北京今天会下雨吗"
- "查询纽约未来3天天气"
- "适合户外运动吗"

## Installation
```bash
npx clawhub@latest install weather
```

## API
使用OpenWeatherMap或其他天气API，需要配置API Key。
