# Tavily Search Skill

## Description
AI-optimized search engine that provides curated, summarized answers instead of just links. Designed specifically for AI agent use.

## When to Use
- Need quick research on any topic
- Want summarized information without reading multiple sources
- Tracking news on specific topics
- Deep analysis requiring comprehensive information

## Key Features
- Direct answers (not just links)
- News tracking with --topic flag
- Deep analysis mode
- 1000 free searches/month

## Setup
Get API key from https://tavily.com (free tier available)

## Usage Examples
- "Search for latest AI news"
- "Deep research on quantum computing"
- "Track news topic: climate change"
