# Browser Automation Skill

## Description
Rust-based headless browser for fast web automation. Enables agents to navigate websites, fill forms, click buttons, and extract data.

## When to Use
- Web scraping and data extraction
- Automated form submission
- Login and session management
- Multi-step web workflows

## Key Features
- Fast Rust-based engine
- Headless and headed modes
- Session persistence
- Element identification and interaction

## Usage Examples
- "Open https://example.com and click the login button"
- "Fill the search form and submit"
- "Extract all product titles from this page"
