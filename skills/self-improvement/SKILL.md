# Self-Improvement Skill

## Description
Records errors and learnings to continuously improve agent performance. Promotes recurring patterns to project memory.

## When to Use
- After making mistakes
- When discovering better methods
- For accumulating best practices
- To avoid repeating errors

## Key Features
- Error logging to .learnings/ERRORS.md
- Learning capture to .learnings/LEARNINGS.md
- Pattern promotion after 3 occurrences
- Best practice documentation

## Usage Examples
- "I made a mistake, let's record this"
- "This is a better way to do it"
- "What have we learned about this?"
