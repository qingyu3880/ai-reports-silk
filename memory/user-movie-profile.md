# 用户电影口味档案

## 印象深刻的电影

### 《法式火锅》(La Passion de Dodin Bouffant, 2023)
- **导演**：陈英雄 (Tran Anh Hung)
- **国家**：法国/越南
- **类型**：美食、爱情、生活哲学
- **用户偏好信号**：
  - 喜欢慢节奏、注重细节的美学
  - 对"日常仪式感"和"生活哲学"有共鸣
  - 欣赏将技艺（烹饪）升华为艺术的表达
  - 对法国文化/生活方式有兴趣

### 《酒精计划》(Druk, 2020)
- **导演**：托马斯·温特伯格 (Thomas Vinterberg)
- **国家**：丹麦
- **类型**：剧情、喜剧、存在主义
- **用户偏好信号**：
  - 喜欢北欧电影的内敛和深度
  - 对"中年危机""人生意义"等主题有共鸣
  - 欣赏有哲学探讨但不沉重的叙事
  - 喜欢麦斯·米科尔森的表演

## 口味画像

**审美倾向**：
- 法国：精致、仪式感、生活美学
- 北欧：内敛、存在主义、人性深度
- 日本：待补充

**主题偏好**：
- 生活哲学 > 戏剧冲突
- 人物内心 > 情节推进
- 视觉美学 > 特效场面

**避雷区**：
- 好莱坞商业大片
- 纯娱乐性/爆米花电影
- 过于沉重压抑的悲剧

## 推荐策略

1. **法国方向**：继续挖掘陈英雄、侯麦、阿巴斯等导演的作品
2. **北欧方向**：托马斯·温特伯格、鲁本·奥斯特伦德、约阿希姆·提尔
3. **日本方向**：是枝裕和、滨口龙介、黑泽清（现代日本）
4. **交叉点**：东西方文化交融的电影（如《法式火锅》导演陈英雄是越南裔法国人）
