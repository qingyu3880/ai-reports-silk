# AI日报生成模板与规范

## 发送格式（必须严格遵守）

### 开头格式
```
以下是2026年X月X日AI日报（已去重）：

---

# AI日报 | 2026年X月X日

> 本日报涵盖全球AI行业最新动态，已自动去重，仅包含今日全新内容。
```

### 内容结构
1. **头条新闻**（5-6条，去重后）
   - 标题
   - 完整原文详情（不少于100字）
   - 参考来源链接

2. **中文AI圈动态**（2-3条，去重后）
   - 标题
   - 完整原文详情
   - 参考来源链接

3. **AI产品与应用**（2-3条，去重后）
   - 标题
   - 完整原文详情
   - 参考来源链接

4. **重要开源项目**（表格形式，去重）
   - 项目名称、组织、描述、GitHub链接

5. **核心洞察**（3条，与历史角度不同）
   - 深度分析，不重复过去7天的观点

### 结尾格式
```
---

*日报生成时间：2026年X月X日*  
*数据来源：公开新闻报道、GitHub Trending等*  
*去重范围：过去7天日报*
```

## 去重规则

1. **标题去重**：新闻标题相似度>80%视为重复
2. **内容去重**：同一事件的不同报道视为重复
3. **时间范围**：检查过去7天的日报文件
4. **开源项目**：同一项目7天内不重复推荐
5. **核心洞察**：分析角度与过去7天不重复

## 无新闻处理

如果今日新新闻少于5条：
```
以下是2026年X月X日AI日报（已去重）：

今日AI圈暂无重大新动态（已检查过去7天日报，未发现全新内容）。

建议关注：
1. [持续跟踪中的话题1]
2. [持续跟踪中的话题2]
```
